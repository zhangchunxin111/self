知识技能

掌握了html和css的基本知识，会移动端布局，响应式布局，栅格系统。

html

超级文本标记语言是 标准通用标记语言下的一个应用，也是一种规范，一种 标准， 它通过标记符号来标记要显示的网页中的各个部分。

css

CSS（层叠样式表）是一种用来表现HTML或XML等文件样式的计算机语言。能够对网页中元素位置的排版进行像素级精确控制

移动端布局

修改视口：把物理像素改成逻辑像素；引入js文件；修改设计稿的宽度，一般以百分比布局为主，以rem布局为辅

响应式布局

响应式布局，就是一个页面能够匹配多个终端，而不是为每个终端做一个特定的版本（pc,手机，平板）

删格系统

用预定义的类来实现响应，更快速的实现响应式布局，栅格系统的使用，可以让网页的信息呈现更加美观易读，更具可用性。

ps

photoshop，具有图像扫描、编辑修改、图像制作、广告创意和图像输入与输出等功能，可以对图像做放大、缩小、旋转等各种变换。
